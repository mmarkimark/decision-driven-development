# Glossary

| Term | Meaning |
|------|---------|
| Decision | Governing architectural record |
| Use Case | Functional requirement derived from a Decision |
| Fast-Lane | Controlled path for rapid operational change |
| Governance | System of rules controlling evolution |
| Domain | Conceptual authority layer |
| Application | Execution/orchestration layer |
| Infrastructure | Tooling and automation layer |
| Traceability | Ability to link changes to governing artifacts |
